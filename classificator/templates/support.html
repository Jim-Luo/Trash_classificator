Bootstrap

The front-end development framework I chose for this project is Bootstrap. My reasons are very simple. Firstly, it is supported by Twitter Co. so that we don't need to worry about the quality and safety of the Framework. Secondly, unlike VueJS or React, Bootstrap is much lighter and suitable for small projects like this. Thirdly, since it exists for years and the community is well-organized, it is easy for us to figure out problems when meeting them.

Django

I chose a Python-based back-end development framework, Django. I have two reasons to choose this framework. First, our machine learning framework is based on Python so it is natural to use a Python-based back-end framework. Second, Python development is very prevelent and we can find enriched resources about Python.
